Summary: Latitude vs. X

Visualizations

This project used OpenWeather API in order to examine over 500 cities and their weather conditions. The goal of this project was to analyze weather patterns in relation to distance from the equator.

After using the API to obtain desired data, the data was cleaned and plotted using matplotlib (pandas package). Cloudiness, temperature, wind speed, and humidity were analyzed in relation to latitude. This website displays the analyzed data and plots.
